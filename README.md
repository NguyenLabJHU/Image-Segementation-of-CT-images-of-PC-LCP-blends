# Image-Segmentation-of-Liquid-Crystalline-Polymer-and-Polycarbonate-Blends
Matlab script used for image processing for cross-sectional slices obtained from micro-CT characterization of PC/LCP blends with varying blend ratios and processed with 0, 1, and 2 LMEs.

Raw images and processed volume figures are available in 

Jiang, Y., Olanrewaju, R., Kazmer, D., & Nguyen, T. D. (2026). Micro-CT Characterization of the Domain and Flaw Morphology of Immiscible Blends: Application to a Liquid Crystalline Polymer/Polycarbonate System processed with Layer Multiplying Elements [Data set]. Zenodo. https://doi.org/10.5281/zenodo.19145055
